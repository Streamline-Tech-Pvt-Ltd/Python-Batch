The Operators are symbols used to perform operations on variables and values.

*Types of Operators in Python:

1.Arithmetic Operators:-
Arithmetic operators are used to perform mathematical calculations.

Types:-

1)Addition(+):
The addition operator is used to add two numbers and return their sum.
 ex.
	a = 10
	b = 5
	print(a + b)
output:15

2.Subtraction(-):
The subtraction operator is used to subtract one number from another.
 ex.
	a = 10
	b = 5
	print(a - b)
output:10

3.Multiplication(*):
The multiplication operator is used to multiply two numbers.
 ex.
	a = 10
	b = 5
	print(a * b)
output:50


4.Division(/):
The division operator divides one number by another.
 ex.
	a = 10
	b = 5
	print(a / b)
output:2


5.Modulus( " % " Remainder) :
The modulus operator returns the remainder after dividing one number by another.
 ex. 
	a = 10
 	b = 3
	print(a % b)

6.Exponent (Power " ** "):
The exponent operator is used to calculate the power of a number.
  ex.
	a = 2
	print(a ** 3)
output:8

7.Floor Division (Quotient " // "):
The floor division operator divides two numbers and returns only the integer part of the result.

 ex.
	a = 10
	b = 3
	print(a // b)
output:3

4)Logical Operators:-
Logical operators are used to combine two or more conditions and They will be return a Boolean value True or False.
it has three logical operators:

1.AND:
The and operator returns True only if both conditions are True and If any one condition is False, the result will be False.
 ex.
	a = 10
	b = 5
	print(a > 5 and b > 2)	
output: True

2.OR:
The or operator returns True if at least one condition is True and It returns False only when both conditions are False.
 ex.
	a = 10
	b = 5
	print(a > 20 or b > 2)
output: True

3.NOT:
The not operator is used to reverse the result of a condition. If condition is True not makes it False. If condition is False not makes it True.
 ex.
	a = 10
	print(not(a > 5))
output: False


5)Identity Operators:-
Identity operators are used to check whether two variables refer to the same object in memory.
Python has two identity operators:
1.is:
The " is " operator returns True if both variables refer to the same object if they refer to different objects, it returns False.
 ex.
	a = 10
	b = 10
	print(a is b)
output: True

2.is not:
The " is not " operator returns True if both variables refer to different memory locations.
 ex.
	a = [1, 2]
	b = [1, 2]
	print(a is not b)
output: True

6.Membership Operators:-
Membership operators are used to check whether a value exists in a sequence like List, Tuple, String, Set.
Python has two membership operators:
1.in:
The " in " operator returns True if the specified value is present in the sequence.
 ex.
	num1 = [10, 20, 30]
	print(20 in num1)
output: True

2.not in:
The not in operator returns True if the specified value is NOT present in the sequence.
 ex.
	num2 = [10, 20, 30]
	print(10 not in num2)
output: False



7.Bitwise Operators:-
Bitwise operators are used to perform operations on binary numbers and Computers store numbers in binary form (0 and 1).

Types:-

1.AND(&):
The & operator returns 1 only if both bits are 1.
 ex.
	a = 5      # Binary: 101
	b = 3      # Binary: 011
	print(a & b) 
output: 1	# Binary:001

2.OR(|):
The | operator returns 1 if at least one bit is 1.
 ex.
	a = 5      # 101
	b = 3      # 011
	print(a | b)
output:7	#Binary:111

3.XOR(^):
The ^ operator returns 1 if both bits are different.
 ex.
	a = 5      # 101
	b = 3      # 011
	print(a ^ b)
output:6	#binary:110

4.NOT(~):
The ~ operator inverts all bits (0 becomes 1, 1 becomes 0).
formula:-
	~a=-(a-1)
 ex.
	a = 5
	print(~a)
output:-6

*Conditional Statements (if ,elif ,else):
This are used to make decisions in a program and They can execute different blocks of code based on conditions.

Python provides:
1) if:
The if statement is used to test a condition. if the condition is True the block of code inside the if statement is executed. if the condition is False the block is skipped.

ex.
	age = 20
	if age > 18:
    	   print("You are eligible to vote")


2) elif:

The elif statement stands for “else if” and It is used to check another condition if the previous if condition is False. When it finds the first True condition, It executes that block, Then skips the remaining conditions.

ex. 
	marks = 60
	if marks > 75:
   	   print("Grade A")
	elif marks > 50:
  	     print("Grade B")

3) else:

The else statement is used when previous conditions are False and it acts as a default block of code.
 
ex. 
	
	marks = 60
	if marks > 75:
    	   print("Grade A")
	else:
   	    print("Fail")


*Transfer Statements:-
it will be used to control loop execution 

1.break:
The " break " statement is used to stop the loop execution immediately.
 ex.
	for i in range(1, 6):
    	    if i == 4:
               break
    	    print(i)
output: 1, 2, 3

2.continue:
The continue statement is used to skip the current iteration and move to the next iteration of the loop and It does not stop the loop completely.
 ex.
	num=[1,2,3,4,5,6]
	if num == 3:
	   continue
	print(num)
output: 1, 2, 4, 5, 6

3.pass:
It is used as a placeholder when a statement is required syntactically but no code is needed.
 ex.
	for i in range(3):
   	   pass
	print(i)
output:2

A function is a block of code that runs only when it is called.
It helps:
1.Avoid repetition
2.Make code reusable
3.Make program organized

syntax:
	
	def function_name(parameters):   # Function Definition
    		# function block / logic
    		return value

	variable = function_name(arguments)  # Function Calling


ex.
	def addition(num1, num2):
   	    return num1 + num2

	print(addition(10,20))
	print(addition(30,40))
	print(addition(50,60))

*Scope in Python:-
Scope means where a variable can be accessed.

*Types of Scope:
1. Global Scope:
the global scope Declared outside function.
ex.
age = 23  # Global variable
Accessible everywhere in program.

2. Local Scope:
the local scope Declared inside function.
ex.
	def main():
	   name = "Krishna"  # Local variable

Accessible only inside that function.

*Namespaces in Python:
Namespace : A place where variable names are stored.

*Types:
1. Built-in Namespace:
Examples: print(), len(), type()

2.Global Namespace:
ex.
	x = 5

3. Local Namespace:
ex.
	def main():
    		   x = 5

4. Enclosing Namespace (Nested Function):
ex.

	def function1():
    		   x = 5   # Enclosing namespace

    		def function2():
       		    y = 10
      		print(y)

    			function2()
  		print(x)
	function1()

output:
10
5

*Arguments in Python:
Argument: Actual value passed to function.

*Types of Arguments:-
1.Positional Argument:
Order matters.
ex.
	def student(name, rollno, age):
   		   return f"My name is {name}. My roll no is {rollno} & my age is {age}"

	print(student("Pramod", 12, 21))

2.Keyword Argument:
the Order does NOT matter.
ex.
	def student(name, rollno):
    		   return f"My name is {name}. My roll no is {rollno}"

	print(student(rollno=12, name="Pramod"))

3.Default Argument:
the Default value is given in function definition.
ex.
	def student(name, age=21):
    		   return f"My name is {name}. My age is {age}"
	print(student("Pramod", 23))

If age not passed:

	print(student("Pramod"))

Output:  My name is Pramod. My age is 21
